Raqaypampa Data

This folder contains clean and raw data from surveys, interviews and timeseries regarding electricity usage in the Raqaypampa region, Bolivia.

⚠️ Mandatory Attribution & Data Status

Status: Pre-publication / In Press.

This dataset is part of an ongoing research project. If you intend to use this data for any analysis, visualization, or publication, you are strictly required to reference the following source:

    Claudia Sanchez-Solis, Sergio Balderrama, Jaime Zambrana, Sylvain Quoilin, "Modeling Household Behavior in the Context of Solar Home System Electrification: A Case Study in Raqaypampa, Bolivia", 2025. (In Press / Forthcoming).

Important: As this research is not yet officially published, we kindly request that you contact the lead researcher [clsanchez@uliege.be) before submitting any work that utilizes this dataset to ensure there is no overlap with the primary findings of the ARES-funded project.

Acknowledgements

This data was collected and processed as part of the project "Tailored Energy Systems for Energy Planning in Bolivia." We gratefully acknowledge the financial support and partnership of ARES (Académie de Recherche et d'Enseignement Supérieur) in making this research possible.
